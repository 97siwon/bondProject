  <h3>✍️Description✍️</h3>
  BOND는 같은 관심사를 가진 수많은 사람들이 모여 모임을 만들고 <br>
  모임에 글과 이미지를 공유하고 그에 대한 생각과 정보를 나누며 서로 이야기를 나누는 공간, <br>
  일정 기능을 통해 모임의 정모를 정할 수 있는 공간입니다. 
	
  <br>
  <br>

  <h6>	1. 만들기 버튼 클릭 시 새 모임 만들기 페이지로 이동

        2. 내 모임 페이지에는 사용자가 가입한 모임 리스트만 출력됨 

        3. 사용자가 가입한 모임의 일정이 출력됨

        2. 사용자 권한이 리더일 경우,밴드 설정 버튼 사용 가능

        3. 게시물은 무한 스크롤 기능을 이용하여 스크롤 시 마지막 게시물까지 출력됨
  </h6>
  <br>

  <br>
    <img width=50% src="https://user-images.githubusercontent.com/110653581/206651992-69558248-55da-433d-9e2d-640158020e29.PNG"/>
  <h5>👆멤버 페이지👆</h5>
  <h6>	1. 가입한 멤버 리스트 출력

        2. 사용자가 맨 첫 줄에 출력되고, 두번째줄에 리더가 출력, 그 외 사용자는 가나다 순으로 출력됨.

        3. 다른 사용자 신고 기능

        4. 멤버 초대하기 클릭 시 모임 가입 주소 복사됨
  </h6>

  <h5>👆일정 페이지👆</h5>
  <h6>	1. 가입한 모임 일정 관리

        2. 일정 만들기 버튼 클릭 시 모달 창으로 만들기 창 출력
         (일정 제목, 설명, 날짜 설정 가능)

        3. 캘린더의 일정 클릭 시 모달창으로 일정 상세 정보 출력
  </h6>
  <br>

  <h5>👆모임찾기 페이지👆</h5>
  <h6>	1. 주제별 카테고리 클릭 시 주제별 모임 리스트가 보이는 화면으로 이동

        2. 검색 기능을 통해 사용자가 찾고자하는 모임 검색 가능
           (본드 이름/본드 소개글)

        3. 모임 추천 영역에서 사용자가 가입하지 않은 모임 추천
